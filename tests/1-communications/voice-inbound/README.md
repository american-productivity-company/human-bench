# Voice Inbound Communication Tests

Tests ability to answer and handle incoming phone calls.

## Weight: 20% of Communications (2% of total HLI)

## What to Test

- Call answering and greeting
- Voice clarity and natural speech
- Active listening and comprehension
- Conversation flow and turn-taking
- Graceful call conclusions
- Handling interruptions and clarifications

## Test Examples

- Basic greeting and introduction
- Quick status checks
- Information requests via call
- Multi-turn conversations
- Handling unclear speech or background noise

## File Naming

- `greeting-basic.yaml` - Simple call answering
- `status-check.yaml` - Quick information requests
- `multi-turn-conversation.yaml` - Extended dialogs
- `clarification-handling.yaml` - Dealing with unclear input
- `graceful-endings.yaml` - Call conclusion tests

## Pass Criteria

- Call connected within 30 seconds
- Clear greeting with identification
- Natural conversation pacing
- Appropriate use of verbal confirmations ("uh-huh", "I understand")
- Clean call ending without abrupt disconnection
