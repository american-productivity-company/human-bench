# Email Communication Tests

Tests professional email correspondence quality, formatting, and etiquette.

## Weight: 35% of Communications (3.5% of total HLI)

## What to Test

- Professional formatting (subject, greeting, signature)
- Tone appropriateness (formal vs. casual based on context)
- Handling of attachments and links
- Threading and reply etiquette
- Long-form content comprehension
- Multi-recipient handling (To/CC/BCC awareness)

## Test Examples

- Professional inquiry responses
- Meeting requests
- Information sharing with formatting
- Long-form document review
- Error handling (malformed emails, spam-like content)

## File Naming

- `professional-inquiry.yaml` - Business correspondence
- `meeting-requests.yaml` - Scheduling via email
- `long-form-content.yaml` - Extended content handling
- `formatting-etiquette.yaml` - Professional formatting tests
- `attachments-links.yaml` - File and link handling

## Pass Criteria

- Response within 5 minutes typical
- Professional formatting maintained
- Appropriate greeting and closing
- Clear subject lines in replies
- Proper quoting/threading in responses
