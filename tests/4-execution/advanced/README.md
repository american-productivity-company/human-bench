# Advanced Execution Tests

Extended tasks requiring adaptive strategy and multi-stakeholder coordination.

## Weight: 30% of Execution (12% of total HLI)
## Multiplier: 2.0x

## Characteristics
- Multi-day extended tasks
- High ambiguity
- Proactive problem-solving
- Error detection and recovery
- Adaptive strategy
- Multiple stakeholder coordination

## Examples
- Client onboarding (sales → legal → success)
- Plan quarterly offsite (logistics + agenda + follow-up)
- Product launch coordination
- Manage project from start to finish
- Complex multi-team initiative

## Test Duration
- Expected: Hours to days
- Timeout: Task-dependent

## File Naming
- `client-onboarding-e2e.yaml`
- `offsite-planning.yaml`
- `product-launch.yaml`
- `project-management.yaml`

## Pass Criteria (threshold: 0.85)
- Comprehensive planning
- Proactive issue identification
- Strong stakeholder management
- Adaptive to changing requirements
- High-quality deliverables
- Clear communication throughout
