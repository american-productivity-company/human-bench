# Moderate Execution Tests

Multi-step tasks requiring coordination with clear paths to completion.

## Weight: 25% of Execution (10% of total HLI)
## Multiplier: 1.2x

## Characteristics
- 2-4 sequential steps
- Some decision-making
- Clear success criteria
- May involve coordination
- Single domain/context

## Examples
- Schedule meeting + send agenda
- Book travel + send confirmation
- Update CRM after call
- Generate and send report
- Coordinate single deliverable

## Test Duration
- Expected: 5-15 minutes
- Timeout: 30 minutes

## File Naming
- `meeting-with-agenda.yaml`
- `travel-booking.yaml`
- `crm-update-flow.yaml`
- `report-generation.yaml`

## Pass Criteria (threshold: 0.75)
- All steps completed
- Logical sequencing
- Good communication
- Handles minor issues
