# Simple Execution Tests

Single-step, unambiguous tasks with clear success criteria.

## Weight: 10% of Execution (4% of total HLI)
## Multiplier: 1.0x

## Characteristics
- One clear action
- Immediate execution
- Minimal context needed
- Single correct approach

## Examples
- Set a reminder
- Send a message
- Look up calendar
- Create single event
- Simple API call

## Test Duration
- Expected: < 5 minutes
- Timeout: 10 minutes

## File Naming
- `reminders-basic.yaml`
- `send-message.yaml`
- `calendar-lookup.yaml`
- `single-action.yaml`

## Pass Criteria (threshold: 0.7)
- Task completed correctly
- Confirmation provided
- Reasonable response time
