# Complex Execution Tests

Tasks requiring judgment, synthesis, and cross-domain coordination.

## Weight: 35% of Execution (14% of total HLI)
## Multiplier: 1.5x

## Characteristics
- 5+ interdependent steps
- Significant judgment required
- Multiple valid approaches
- Cross-domain coordination
- Information synthesis
- Ambiguity resolution

## Examples
- Research competitors + summary
- Coordinate multi-timezone schedules
- Prepare for board meeting
- Analyze data + recommendations
- Multi-system integration task

## Test Duration
- Expected: 15-60 minutes
- Timeout: 2 hours

## File Naming
- `competitor-research.yaml`
- `schedule-coordination.yaml`
- `meeting-preparation.yaml`
- `data-analysis.yaml`

## Pass Criteria (threshold: 0.8)
- Demonstrates judgment
- Synthesizes information
- Handles ambiguity well
- Produces quality output
- Good stakeholder communication
